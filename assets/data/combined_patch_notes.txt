- Arcade_Evolution — 2026-04-03
- What it does
- Scans the folder this script is placed in by default, including subfolders.
- Detects patch note .txt files by common names like patch, update, changelog, and release notes.
- Maps each patch file to the nearest app folder that contains a main.py.
- Treats the scan root as the main directory and forces that section to the top of the combined export.
- Sorts entries newest first.
- Removes duplicate patch note files from the final export.
- Exports one clean combined .txt file without file paths inside the output text.
- Can optionally delete older patch note files while keeping the newest N per app.

- How to run
- Gui:
- python patch_notes_workstation.py

- Cli:
- python patch_notes_workstation.py --cli --root "." --output "combined_patch_notes.txt"

- Optional cleanup:
- python patch_notes_workstation.py --cli --root "." --output "combined_patch_notes.txt" --delete-older --keep-latest-per-app 1

- Notes
- No external dependencies are required beyond Python with Tkinter available.
- The GUI defaults to scanning the script's own directory if no config file exists.
- The preview image included with this package is a rendered workstation preview based on live scan data from testing.

Arcade_Evolution (Main Directory)
=================================

Arcade_Evolution — 2026-04-03
-----------------------------
- What it does
- Scans the folder this script is placed in by default, including subfolders.
- Detects patch note .txt files by common names like patch, update, changelog, and release notes.
- Maps each patch file to the nearest app folder that contains a main.py.
- Treats the scan root as the main directory and forces that section to the top of the combined export.
- Sorts entries newest first.
- Removes duplicate patch note files from the final export.
- Exports one clean combined .txt file without file paths inside the output text.
- Can optionally delete older patch note files while keeping the newest N per app.

- How to run
- Gui:
- python patch_notes_workstation.py

- Cli:
- python patch_notes_workstation.py --cli --root "." --output "combined_patch_notes.txt"

- Optional cleanup:
- python patch_notes_workstation.py --cli --root "." --output "combined_patch_notes.txt" --delete-older --keep-latest-per-app 1

- Notes
- No external dependencies are required beyond Python with Tkinter available.
- The GUI defaults to scanning the script's own directory if no config file exists.
- The preview image included with this package is a rendered workstation preview based on live scan data from testing.
